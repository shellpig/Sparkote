## 敘事定位（D1，第一封信）

- 大千第一個正式區域，起始霧區出口進入。
- 含**回音水塘（事件點）**：乾枯水塘、中央立生鏽的「祈雨鐘」。
- 玩家在林中採**木材**，修好敲鐘的木槌（資源修復），敲響祈雨鐘，把無字雨聲信投入鐘底石縫。
- 送達後：天降大雨 → 融化靜止濃霧（消散霧區）→ 大片森林顯影 → 水塘邊長出晶石（第二世界鋪墊）。
- 回禮：「雨滴風鈴」（第一個家園家具）。
- 結尾鉤：雨滴風鈴夾著一小片來自第二世界的晶石光。

## 含資源 / 事件

- 木材資源點（`type=resource_point`，產出 wood；首採 / 重採見 `subdocs/_global/resources.md`）。
- 回音水塘事件（`type=event`）：修木槌 → 敲鐘 → 送信 → 世界復甦，連動雨滴風鈴獎勵與第二世界鋪墊。
